Execução Manual do Job

O job de monitoramento é executado automaticamente toda hora pela Cloud Function agendada.
Use o botão abaixo para disparar manualmente durante testes ou sob demanda.

O que o job faz:
  • Busca todos os produtos ativos no Firestore
  • Faz a busca de cada produto na API do Mercado Livre
  • Salva snapshot de preços no histórico (price_history)
  • Detecta ofertas por desconto, novo mínimo ou promoção ativa
  • Salva ofertas detectadas na coleção deals

Agendamento Automático

A Cloud Function priceMonitorJob está agendada para rodar toda hora no fuso de Brasília.
Para alterar a frequência, edite functions/index.js e faça um novo deploy.

Atenção: Cloud Functions com chamadas a APIs externas e Scheduled Functions requerem o plano Blaze (pay-as-you-go) do Firebase. O uso dentro dos limites gratuitos mensais não gera cobrança.
